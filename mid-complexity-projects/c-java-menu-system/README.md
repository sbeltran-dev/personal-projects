# C & JAVA MENU SYSTEM

An interactive menu system in C that can execute Java programs.

## What this program does?

It displays a menu with 5 options to execute Java exercises (Fibonacci_Sequence, Fractions_Summation, etc.).

## Requirements

- C compiler (GCC).
- JDK installed.
- Make sure that the exercises and the commands to execute the Java exercises are the same.

## How to use it

1. Enter these commands in the terminal: 
- Compiling the C file:
```
   gcc MenuProject.c -o MenuProject
```

- To run the program:
```
   ./MenuProject
```

2. When the menu displays, just select one of the 5 main options, or 0 to leave the program.
